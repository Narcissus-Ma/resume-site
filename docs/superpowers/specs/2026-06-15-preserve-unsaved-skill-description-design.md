# 保留未保存技能描述设计

## 问题

动态列表输入框由 Ant Design Form 管理，新增按钮却基于 React 状态中的旧 `data` 追加内容。调用 `updateData` 后，旧数据会重新写回 Form，导致用户尚未保存的输入被覆盖。受影响操作包括添加技能描述、技能类别、技能、工作经历、项目经历、教育经历和网站链接。

## 方案

执行动态列表新增操作时，先通过 `form.getFieldsValue(true)` 获取表单实时值，并以它作为当前简历数据，再追加对应默认项。合并与追加逻辑放入纯规则函数，Hook 只负责读取 Form 和更新状态。

## 测试

回归测试构造旧数据和包含未保存输入的表单数据，分别验证所有新增操作在追加后保留表单输入。
